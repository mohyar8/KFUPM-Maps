# KFUPM Maps

KFUPM Maps is a web-based navigation application designed to help students and visitors navigate the KFUPM campus through **photo-based routes**. Instead of traditional map directions, the application provides step-by-step images with descriptions to guide users between campus buildings.

## Features

- Search for routes between campus buildings.
- View step-by-step photo navigation.
- User authentication (Sign Up / Login).
- Submit and manage custom routes.
- Leaderboard based on user contributions.
- Admin approval system for submitted routes.
- User profile management.

## Technologies Used

### Frontend
- React (Vite)
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
npm run server
```

### Backend

```bash
cd backend
npm install
npm run dev
```

The goal of this project is to develop a full-stack web application that helps users navigate the KFUPM campus through photo-based routes while applying modern web development practices, database management, authentication, and RESTful API design.
